اتصال حرفه‌ای WordPress به سیستم CRM با پشتیبانی کامل از Multi-Tenant و WooCommerce

== Description ==

افزونه WordPress CRM Integration - Professional یک راه‌حل کامل و حرفه‌ای برای اتصال وب‌سایت WordPress شما به سیستم CRM است. این افزونه به‌طور خودکار اطلاعات مشتریان، سفارشات WooCommerce و کاربران را با سیستم CRM همگام‌سازی می‌کند.

**ویژگی‌های کلیدی:**

* **همگام‌سازی خودکار**: ثبت‌نام کاربران جدید و به‌روزرسانی‌ها به‌طور خودکار به CRM ارسال می‌شود
* **پشتیبانی Multi-Tenant**: هر tenant کلید API مجزا و داده‌های جداگانه دارد
* **یکپارچگی WooCommerce**: سفارشات، مشتریان و اطلاعات صورتحساب به‌طور کامل پشتیبانی می‌شود
* **رابط مدیریت فارسی**: رابط کاربری کاملاً فارسی و RTL
* **سیستم لاگ پیشرفته**: ثبت کامل تمام فعالیت‌ها و خطاها
* **همگام‌سازی دستی**: امکان همگام‌سازی دستی کاربران موجود
* **تست اتصال**: بررسی آسان اتصال به CRM
* **صف همگام‌سازی**: پردازش ایمن و قابل اعتماد داده‌ها

**سازگاری:**

* WordPress 5.0 یا بالاتر
* PHP 7.4 یا بالاتر
* WooCommerce (اختیاری)
* تمام تم‌های استاندارد WordPress

== Installation ==

1. فایل افزونه را در پوشه `/wp-content/plugins/wordpress-crm-integration` آپلود کنید
2. افزونه را از منوی 'افزونه‌ها' در WordPress فعال کنید
3. به منوی 'CRM Integration' بروید و تنظیمات را انجام دهید
4. آدرس CRM، کلید API و کلید Tenant را وارد کنید
5. اتصال را تست کنید و همگام‌سازی را فعال کنید

== Frequently Asked Questions ==

= آیا این افزونه با WooCommerce کار می‌کند؟ =

بله! افزونه به‌طور کامل با WooCommerce یکپارچه شده و سفارشات، مشتریان و اطلاعات صورتحساب را همگام‌سازی می‌کند.

= چگونه کلید API دریافت کنم؟ =

کلید API را از پنل مدیریت سیستم CRM خود دریافت کنید. هر tenant کلید API مجزا دارد.

= آیا اطلاعات قدیمی همگام‌سازی می‌شود؟ =

افزونه فقط کاربران جدید و به‌روزرسانی‌ها را به‌طور خودکار همگام‌سازی می‌کند. برای همگام‌سازی کاربران موجود از قسمت "همگام‌سازی دستی" استفاده کنید.

= آیا افزونه ایمن است؟ =

بله، افزونه از رمزگذاری HTTPS و احراز هویت API استفاده می‌کند. تمام اطلاعات به‌صورت امن منتقل می‌شود.
